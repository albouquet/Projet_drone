Ce dossier contient tous les éléments pour la détéction de mouvement.

Le programme python test_opcv.py récupère des snapshots (frames) depuis la vidéo vid1 (réalisé avec la raspberry pi).
Il soustrait les 2 frames (récupérées à 1 seconde d'intervalle)  afin de mettre en évidence les différences entre les deux.
